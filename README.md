Dot files
=========

# How to use
Get repo

```
$ git clone https://github.com/myyang/dot_file.git
$ cd dot_file
```

Run setup dot file script

```
$ ./setup.sh
```

If dev-tools(see next topic for details) are needed, run

```
$ ./setup_dev_env.sh
```

# Dev-tools list

* Homebrew
* git
* git-flow
* pip
* bash-completion
